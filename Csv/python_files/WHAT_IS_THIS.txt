app/routers/stock_csv_router.py
→ HTTP 요청을 받아서 응답을 돌려주는 문지기


app/services/stock_csv_service.py
→ 실제 CSV 처리와 DB 로직을 수행하는 작업자


app/utils/csv_inspector.py
→ CSV 파일 분석기(Inspector)
CSV 업로드의 사전 진단기라고도 할 수 있음.

1. 잘못된 인코딩, 헤더, 구분자 등을 자동 감지/정규화
2. 서비스 로직(stock_csv_service.py)이 파일 내용 검증·DB 반영에만 집중할 수 있게 해줌
3. 파일 오류나 포맷 차이로 인한 업로드 실패율을 낮춤


+ 드라이런 기능 추가 (★중요)
→ 시뮬레이션 기능
(DB에 직접 데이터를 넣는 방식이 아닌, 시뮬레이션으로 데이터를 넣고
문제가 발생하는지, 정상적으로 작동하는지 등 기능을 테스트+시연)

예상 효과:
사용 방법을 미리 숙지할 수 있고, 
재고 수량 불일치 등의 문제가 발생하기 전에 사전에 방지할 수 있어
업체의 사장님들이 좋아하시는 기능이 될 것.
